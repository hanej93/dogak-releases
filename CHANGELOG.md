# 변경 이력

버전 표기는 [유의적 버전](https://semver.org/lang/ko/)을 따릅니다.

## v0.2.1 (2026-07-25) — 설정창 경량화·안정성 개선

**성능 · 편의**
- 설정창을 별도 헬퍼 프로세스로 분리해 평상시 상주 메모리 사용량 절감
- 메인 앱과 설정창 사이의 청소 모드 상태 동기화

**안정성**
- 설정창 중복 실행, 포커스, 단축키 입력 및 설정 저장 경합 문제 수정
- 빠른 사운드팩 전환 요청이 누락되던 문제 수정
- 첫 트레이 하위 메뉴가 예기치 않게 닫히던 문제 수정

**플랫폼**
- Windows 키 입력 감지, 트레이, 자동 실행, 단일 인스턴스 및 온보딩 기반 추가

> 이 버전도 Apple 서명·공증 전 **비공식 베타**입니다. 설치 시 README의 안내대로 macOS 경고를 한 번 풀어주세요.

## v0.2.0 (2026-07-24) — 사운드팩·자동화 대폭 강화

v0.1.0 이후 사운드팩 생태계와 편의 기능이 크게 넓어졌습니다.

**사운드팩**
- 온라인 사운드팩 레지스트리 — 앱에서 바로 둘러보고 내려받기
- zip / Mechvibes 팩 가져오기
- 팩 카테고리 분류 + 상세정보 패널 + 즐겨찾기
- 마우스 클릭 사운드 지원
- 키맵 편집 GUI — 키별 소리를 직접 손보기

**자동화 · 편의**
- 회의/음악 재생 시 자동 뮤트, 헤드폰 연결 시 자동 켜기, 앱별 제외 목록
- 키보드 청소 모드 (닦는 동안 입력·소리 잠금)
- 로컬 전용 타이핑 통계 (기기 밖으로 안 나감)
- 전역 단축키 확장 — 뮤트뿐 아니라 즐겨찾기 팩 순환 등 액션별 지정
- `dogak://` URL scheme 연동
- 팩 전환 시 시스템 알림, 설정 창 닫기 버튼

**안정성**
- 입력 모니터링 이벤트 탭 자동 복구 — 권한 재허용 후에도 알아서 복귀
- 트레이 메뉴·자동 뮤트 배지·로그인 자동 실행 관련 버그 수정

> 이 버전도 Apple 서명·공증 전 **비공식 베타**입니다. 설치 시 README의 안내대로 macOS 경고를 한 번 풀어주세요.

## v0.1.0 (2026-07-22) — 첫 비공식 베타

macOS용 첫 배포. 기계식 타건음 재생의 핵심 + 편의 기능이 모두 포함됩니다.

**핵심**
- 어떤 앱에서 타이핑하든 기계식 키보드 소리 재생 (listen-only — 다른 앱 입력에 영향 없음)
- 저지연 오디오 엔진 (키다운/키업 개별 소리, 동시 재생)
- 사운드팩 시스템 (내장 팩 + 사용자 팩, 실시간 전환)
- 메뉴바 상주 · 온보딩/설정 창 · 한국어/영어

**사운드 품질**
- 공간 음향 (키 위치에 따라 좌/우에서 소리)
- 피치 변조 (같은 키도 매번 미묘하게 다른 소리)
- 모디파이어/내비게이션 키 소리 제외 옵션

**편의**
- 전역 단축키(뮤트 토글), 로그인 시 자동 실행
- 출력 장치 선택 + 장치별 볼륨 기억
- 오디오 버퍼 크기 · 유휴 절전 설정

> 이 버전은 Apple 서명·공증 전 **비공식 베타**입니다. 설치 시 README의 안내대로 macOS 경고를 한 번 풀어주세요.
