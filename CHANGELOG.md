# 변경 이력

버전 표기는 [유의적 버전](https://semver.org/lang/ko/)을 따릅니다.

## v0.1.0 (2026-07-22) — 첫 비공식 베타

macOS용 첫 배포. 기계식 타건음 재생의 핵심 + 편의 기능이 모두 포함됩니다.

**핵심**
- 어떤 앱에서 타이핑하든 기계식 키보드 소리 재생 (listen-only — 다른 앱 입력에 영향 없음)
- 저지연 오디오 엔진 (키다운/키업 개별 소리, 동시 재생)
- 사운드팩 시스템 (내장 팩 + 사용자 팩, 실시간 전환)
- 메뉴바 상주 · 온보딩/설정 창 · 한국어/영어

**사운드 품질**
- 공간 음향 (키 위치에 따라 좌/우에서 소리)
- 피치 변조 (같은 키도 매번 미묘하게 다른 소리)
- 모디파이어/내비게이션 키 소리 제외 옵션

**편의**
- 전역 단축키(뮤트 토글), 로그인 시 자동 실행
- 출력 장치 선택 + 장치별 볼륨 기억
- 오디오 버퍼 크기 · 유휴 절전 설정

> 이 버전은 Apple 서명·공증 전 **비공식 베타**입니다. 설치 시 README의 안내대로 macOS 경고를 한 번 풀어주세요.
