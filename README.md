# Dogak (도각)

[![GitHub Repo stars](https://img.shields.io/github/stars/hanej93/dogak-releases?style=social)](https://github.com/hanej93/dogak-releases)

어떤 키보드로 타이핑하든 **기계식 키보드 타건음**을 내주는 macOS 백그라운드 앱입니다.

> ⚠️ 현재 **비공식 베타** 배포입니다. Apple 정식 등록(유료 서명) 전이라 설치 시 macOS 경고가 뜨며, 아래 안내대로 한 번만 풀어주면 됩니다.

## 다운로드

**[👉 최신 버전 받기 (Releases)](https://github.com/hanej93/dogak-releases/releases/latest)**

- macOS 12(Monterey) 이상 · Apple Silicon / Intel
- 받은 `Dogak-x.y.z.dmg` 파일로 아래 순서대로 설치하세요.

---

## 설치 방법 (5분)

### 1. 앱을 응용 프로그램에 넣기

1. 받은 **`Dogak-0.1.0.dmg`** 를 더블클릭해 엽니다.
2. 창이 뜨면 **Dogak 아이콘을 옆의 `Applications` 폴더로 드래그**합니다.
3. 그 창은 닫아도 됩니다.

### 2. macOS 경고 풀기 (한 줄 붙여넣기)

정식 등록 전 앱이라 그냥 열면 "확인할 수 없어 열 수 없습니다"가 뜹니다. 아래 한 줄이면 해결돼요.

1. **`터미널`** 앱을 엽니다. (`⌘ + 스페이스` → "터미널" 입력 → 엔터)
2. 아래 줄을 **복사해 터미널에 붙여넣고 엔터**:

```bash
xattr -dr com.apple.quarantine /Applications/Dogak.app
```

3. 아무 메시지 없이 다음 줄이 나오면 성공입니다.

> **이 명령이 뭔가요?** 인터넷에서 받은 파일에 macOS가 붙이는 "검역 딱지"만 떼는 명령입니다. 앱 내용을 바꾸거나 시스템을 건드리지 않고, macOS 경고만 없애줍니다.

### 3. 실행 + 키보드 권한 허용

1. `응용 프로그램` 폴더에서 **Dogak**을 더블클릭해 실행합니다.
2. 화면 오른쪽 위 **메뉴바에 키캡 모양 아이콘**이 생기면 켜진 거예요. (Dock에는 안 뜹니다 — 원래 그런 앱입니다.)
3. 첫 실행 안내에 따라 **"입력 모니터링" 권한을 허용**합니다. 시스템 설정이 열리면 목록에서 **Dogak**을 켜주세요.

> **왜 키보드 권한이 필요한가요?** 타이핑할 때 소리를 내려면 "키가 눌렸다"는 신호를 알아야 하기 때문입니다. **어떤 키를 눌렀는지는 저장하지도, 어디로 보내지도 않습니다** — 소리를 내는 용도로만 씁니다.

이제 아무 데서나 타이핑하면 기계식 키보드 소리가 납니다. 🎉

## 사용법

| 하고 싶은 것 | 방법 |
|---|---|
| 소리 잠깐 끄기/켜기 | 메뉴바 아이콘 **우클릭** (아이콘이 흐려지면 꺼진 상태) |
| 볼륨 · 사운드팩 바꾸기 | 아이콘 **좌클릭** → 메뉴에서 선택 |
| 설정 창 열기 | 아이콘 좌클릭 → **설정…** |
| 앱 완전히 끄기 | 아이콘 좌클릭 → **종료** |

## 문제가 생기면

- **2단계에서 "Dogak.app 없음"**: 아직 1단계(응용 프로그램으로 드래그)를 안 한 거예요. 먼저 옮겨주세요.
- **소리가 안 나요**: 시스템 설정 > 개인정보 보호 및 보안 > **입력 모니터링**에서 Dogak이 켜져 있는지 확인하고, 껐다 켠 뒤 앱을 다시 실행하세요.
- **여전히 "열 수 없습니다"**: 2단계 명령을 다시 실행한 뒤 앱을 우클릭 → **열기** → **열기**.

## 삭제

`응용 프로그램`에서 **Dogak**을 휴지통으로 옮기면 끝입니다. (설정은 `~/Library/Application Support/Dogak/`에 남으며 지워도 됩니다.)
